# The Holy Grail

A completed sentence is an act of intelligibility.

Every act of intelligibility resolves three dimensions at once:

**one operation × one orientation × one terminal mode**

These three dimensions converge into one forced landing. The 54 positions are the complete set of those resolutions.

Each card shows one resolution. The name of the cell is not an added label. It is what those three coordinates force.
